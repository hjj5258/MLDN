# switch
1. 以下是switch的简单范例
```
public class JavaDemo
{
	public static void main(String[] args)
	{
		int num = 1;
		switch(num)
		{
			case 1:System.out.println("1");break;
			case 2:System.out.println("2");break;
			case 3:System.out.println("3");break;
			default:System.out.println("没有满足项");break;
		}
	}
}

```
